Privacy Policy for GTA LeetCode Extension

Last updated: 2024

Data Collection

This extension does NOT collect, store, or transmit any personal user data.

How the Extension Works

The GTA LeetCode extension only operates on leetcode.com domains to:

  • Monitor code submission requests to LeetCode's API
  • Display GTA-style overlays based on submission results
  • Play sound effects for user feedback

Permissions Explanation

  • activeTab: To detect when user is on LeetCode.com
  • host permissions (leetcode.com): To function only on LeetCode domains
  • webRequest: To monitor submission API calls
  • scripting: To inject content scripts for overlay display

Data Storage

No user data is stored locally or transmitted to external servers.
